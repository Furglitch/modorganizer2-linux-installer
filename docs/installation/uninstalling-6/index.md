# Uninstalling v6 Instances

With the release of MO2-LINT v7, support for v6 instances and any previous versions has been removed. If you have any v6 instances that you wish to uninstall, please follow the steps below.

## Transferring Modlists

In order to transfer your modlist from a v6 instance to a v7 instance, first create the v7 instance and set it up as you normally would. Afterwards, copy the `mods`, `overwrite`, and `profiles` folders from the v6 instance directory to the v7 instance directory. This will transfer your mods and sort order to the new instance.

More files may be required to be transferred depending on your specific setup, but the `mods`, `overwrite`, and `profiles` folders are the most commonly needed ones. If you have any custom plugins, scripts, or other files that you wish to keep, make sure to transfer those as well.
