# Game-Specific Instructions

Some games require additional setup or have specific considerations when using MO2-LINT. This section provides game-specific guides to help you get started.

> **Note:** Game-specific guides are community-contributed and may not cover all scenarios. If you encounter issues not covered in these guides, please [open an issue](https://github.com/furglitch/modorganizer2-linux-installer/issues) on GitHub.

## Contributing a Guide

If you've successfully set up MO2-LINT for a game that requires special configuration, consider contributing a guide to help other users. See the [Contributing Guide](../../contributing/) for more information on how to contribute.
